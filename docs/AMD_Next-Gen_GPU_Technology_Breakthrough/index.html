Three Core Technologies

⚡

Radiance Cores

Dedicated Ray Tracing Hardware

How It Works

New dedicated hardware block that takes full control of ray traversal - the most compute-heavy part of ray tracing. Operates independently from shader cores, similar to NVIDIA's RT cores.

Key Benefits

  • Real-time ray & path tracing
  • Frees CPU for geometry & simulation
  • GPU focuses on shading & lighting
  • Significant speed boost from hardware traversal
  • Over 2x RT throughput per CU vs RDNA 3
  • Cleaner, faster, more efficient pipeline
Impact: AMD finally catches up to NVIDIA in ray tracing

🔗

Neural Arrays

Infinity Fabric for AI Processing

How It Works

Interconnect system connecting every compute unit together, allowing direct communication without cache. Allows CUs to process large chunks of screen in one go, like a 'single focused AI engine'.

Key Benefits

  • Connects entire compute unit architecture
  • Direct CU-to-CU communication
  • Whole new level of ML performance
  • Enhanced AI upscaling (FSR, PSSR)
  • Better denoising for ray tracing
  • Superior ML scalability
  • Bigger ML models with less overhead
Impact: Revolutionary AI rendering capabilities

📦

Universal Compression

All-Data Compression System

How It Works

Evaluates and compresses ALL GPU data types (not just textures like current systems). Only essential bytes sent to memory, dramatically reducing bandwidth usage.

Key Benefits

  • Compresses all GPU data types
  • ~25% reduction in bandwidth usage
  • Higher frame rates possible
  • Lower power consumption
  • Enables detailed 4K+ environments
  • Greater overall efficiency
  • More detail without bandwidth increase
Current: Only textures & render targets → New: ALL GPU data types
Impact: Game-changing memory efficiency

Current vs Future Technology

Current Gen (RDNA 3/4)

  • ❌ Ray tracing in shader cores
  • ❌ Limited AI interconnect
  • ❌ Texture-only compression
  • ❌ Cache-dependent communication
  • ❌ Higher memory bandwidth usage

Future Gen (RDNA 5 with new tech)

  • ✅ Dedicated Radiance Cores
  • ✅ Neural Arrays interconnect
  • ✅ Universal compression (all data)
  • ✅ Direct CU-to-CU communication
  • ✅ 25% less bandwidth needed
