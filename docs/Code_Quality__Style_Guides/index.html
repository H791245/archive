‚úÖ What Belongs Where

React: DO Place Here

‚úì Components: UI logic only
‚úì Hooks: Reusable logic
‚úì Styling: Component-level styles
‚úì Tests: Next to components
‚úì Constants: Local configuration

React: DON'T Place Here

‚úó Business logic: Use services instead
‚úó API calls: In custom hooks or services
‚úó State mutation: Immutable updates only
‚úó Multiple rendering functions: Extract to components
‚úó Browser storage: In useEffect, not top-level

Django: DO Place Here

‚úì Services: All write operations
‚úì Selectors: All read/filter operations
‚úì Model methods: Simple properties
‚úì Model clean(): Validation logic
‚úì Tests: In tests/ folder

Django: DON'T Place Here

‚úó Business logic: In Views or Serializers
‚úó Complex queries: In custom managers
‚úó Logic: In model save() method
‚úó Side effects: In Signals
‚úó API routes: Stays separate
